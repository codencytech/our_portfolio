# The Codency - Professional Portfolio Website

## Overview
This is a professional portfolio website for The Codency, a development company founded by two partners. The website features a modern, futuristic design with smooth animations, interactive elements, and a project filtering system.

## Features
- Modern, professional design suitable for a development company
- Fully responsive layout that works on all devices
- Interactive project filtering by category
- Smooth scroll animations and parallax effects
- Contact form with validation
- Animated statistics counter
- Clean, maintainable code structure
